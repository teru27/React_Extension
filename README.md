## 概要
　小説家になろうのテキストを再生するchromeの拡張機能です。

## バージョン

 - node 22
 - react 19.1.1
 - TypeScript 5.9.3
 - vite 7.1.14

## 構成

React + TypeScript + Vite

## 初回導入時

git clone https://github.com/teru27/React_Extension.git

npm i

## ビルド方法

npm run build:all

## 主な機能
- Web Speech APIでの読み上げ機能
- VOICEVOXでの読み上げ機能
